# K-RAINES Skillthon Projects 2026

This repository is the root container for projects, demos, and supporting build artifacts from K-RAINES Skillthon 2026.

## Layout

- `tipa_us_academic_collaborator_agent/`
  TIPA US Academic Collaborator Finder demo app and its planning artifacts.

## Conventions

- Each project should live in its own top-level folder.
- Keep runnable source, tests, and project-specific docs inside that folder.
- Avoid mixing local runtime databases, logs, caches, or machine-specific state into the repo root.
- If a project has setup instructions, keep them in that project's own `README.md`.

## Current Projects
